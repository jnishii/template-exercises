# チェックリスト(unixの課題用)

このチェックリストは，教員が課題の内容確認をするときに使います。

**共通の注意点**

全ての課題について以下を確認すること!

1. 仕様を満たしていること
2. 動作不良? 引数をつけたり，つけなかったり，色々な場合に正常動作するか確認する
こと。何も表示されずに終わることはないように確認すること。
3. 引数が不適切な時にも，きちんとUsageが表示されるようにすること
4. まず引数確認等のエラーチェックをする。その後に本来実行するべき命令を書く
